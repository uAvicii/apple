    崭新创作界面，
    为触控发声。

    有了多点触控功能的加持，iPad 变身表现力十足的专业乐器，敬请上手演奏。搭配各种动态软件乐器，做起音乐得心应手；还有数百款强大插件，操控格外直观顺手。

    动画演示一个人在 iPad Pro 上点按 Logic Pro 中的虚拟乐器，以触控方式创作节拍。

    好声音一触即发，表现力十足的软件乐器任你演奏。弹响虚拟钢琴的琴键，敲击精准发声的鼓垫，你的 iPad 灵活变身各种乐器。

    简洁明了的视图，各种插件的主要控制项都汇于一处，近在指尖。轻轻一扫手指，音轨上的所有插件尽收眼底；轻点两下板块，全部控制项随你取用。

    触控之间，快速直观地浏览大型制作项目。你可以手指开合缩放视图；可在不同区域间拖放；还可轻扫轨道头，显示基本的混音控制项，随手快速调整。

    在 iPad 版 Logic Pro 中打开 iOS 版库乐队的项目，运用各种专业功能和工作流程，释放你的无限创意1。

    拿起 Apple Pencil，进行细致的自动调音和精准的剪辑，让你的音轨焕发新声。加个妙控键盘或键盘式智能双面夹，利用强大的快捷键，
    强力助推你的工作流程2。

    音效翻新，
    灵感翻花样。

    快速寻找并发现新鲜音效，让你的创意源源不断。日益丰富的资源库带来更多乐器、乐段和 Patch，给你层出不穷的灵感。

    你的全部音效在一处尽览。可用的乐器 Patch、音频 Patch、插件、采样和乐段都能快速查看，还能直接从浏览器试听每个音效。

    有了支持联想预测的过滤系统，发掘理想的音效更为轻松。

    直接在 app 内下载免费声音包，即刻拥有不断扩充的素材收藏，尽享丰富的免版税乐段、采样和乐器3。

    专业插件，
    玩得很靠谱。

    数以百计的乐器和音效，让你轻松塑造乐曲的音质特征和音色。

    iPad Pro 展示来自 Alchemy 合成器的多个音频采样可立即在 iPad 版 Logic Pro 中播放。

    使用 Sample Alchemy，动动手指就能将任何音频采样转换成
    可塑的声音供你演奏。

    丰富齐全的高级合成器、复古键盘和逼真的原声乐器，
    让你灵活创作曲风各异的音乐。

    运用独特的混响和延迟，以及复古的均衡器、压缩器和调制效果，精心塑造并修饰音轨的个性风格。

    打起节拍，
    火起来。

    为你的项目编排节拍和低音声部，制作理想的自定鼓组。将任意采样变成手中的新乐器。动动手指，重塑动听的旋律和节奏。

    • Beat Breaker
    • Quick Sampler
    • 步进音序器
    • Drum Machine Designer
    • 实时循环乐段

    iPad Pro 展示使用 iPad 版 Logic Pro 编辑声音采样的时序。

    ♪ 人声混音编曲♪

    借助 Beat Breaker 的时间和音高变形功能，动动手指轻扫或开合，就能彻底重塑或迅速靠齐声音。

    通过切分、翻转或变换，将音频采样或乐段变成能演奏的全新乐器。

    简单轻点几下，就能编排鼓点节奏、低音声部和旋律，甚至实现插件的自动执行。还能运用随机的步进、力度、音符重复等，
    随心添加变奏。

    在任意鼓垫上套用采样和独特的插件，为你的歌曲创建合拍的自定鼓组。Drum Machine Designer 还与步进音序器整合，方便你快速制作节拍。

    抓住创作灵感，动手录音、混音并配搭循环乐段和乐句，快速搞定编曲。轻点各个音效方格，触发循环乐段，然后挑选合拍的组合，实时创作乐曲章节。

    混音，飙出高度。

    通过指尖触控，直接调节推子和其他控制项，就像操作真正的混音控制台一样自如。即使面对充满挑战的混音项目，也能快速完成。

    • Mastering Assistant
    • 专业混音器
    • 触控操作
    • Track Stack

    视频演示一个人正在使用 iPad 版 Logic Pro 的专业混音器功能。

    Mastering Assistant 及其一系列直观易用的专业音效塑造工具在手，你的混音大作随时可供发布。

    功能齐全的混音控制台，包含通道条、音量推子、声相控制、插件、发送，以及精准自动化功能，一切都由你全面掌控。

    直观的多点触控体验，让混音过程妙趣横生。来吧，用指尖触动一下刻度盘，你就懂了。

    将多个相关音轨合并成一个音轨，再用 Track Stack
    功能快速创建子混音。

    各种进阶功能，
    干起活各种专业。

    在 Mac 和 iPad 之间轻松转移你的 Logic Pro 项目，无论在工作室还是出门在外，随时都能上手创作4。

    可在 Logic Pro 项目中直接发掘兼容的第三方乐器和音效，也可从 App Store 下载 Eventide、FabFilter 和 Moog Music 等顶尖
    开发者的 app5。

    使用由 Apogee、Focusrite、Novation、Universal Audio 等主要制造商出品，并且兼容的第三方音频接口、MIDI 接口和控制器，轻松连接
    各种常用设备6。

    与数百款支持 Link 的 app 和设备同步播放速度7。

    通过 compressor 等各种特效，接入来自软件乐器和循环乐段等不同
    来源的音频，创作更具动感和冲击力的混音作品。
